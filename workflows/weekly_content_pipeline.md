# 주간 콘텐츠 파이프라인 — 운영 매뉴얼

## 목적
주 1회 블로그 글 하나를 만들어, 4개 채널에 배포한다.
예산 0원. 콘텐츠 재가공만으로 유입을 만든다.

## 전체 흐름

```
[월] Notion에 블로그 글 작성 (원본)
      ↓
[화] nutriformula.co.kr 자동 반영 (Notion CMS, 이미 구축됨)
      ↓
[수] 네이버 블로그에 같은 글 올림 (하단 광고 삽입)
     구글 검색 노출은 자체 사이트 SEO로 처리
      ↓
[목] 인스타그램 카드뉴스 10장 생성 + 인스타 업로드
      ↓
[금] (향후) 유튜브/인스타 쇼츠 스크립트 → 영상 업로드
```

> 요일은 가이드라인. 사장이 "이번 주 블로그 올려줘"라고 하면 이 순서대로 처리.

## 1단계 — Notion 블로그 작성

**누가:** 사장 또는 Agent(Claude)
**어디서:** Notion → 서박사 영양정보 블로그 데이터베이스
**규칙:**
- 제목: 키워드 포함, 궁금증 유발 (예: "마그네슘 종류별 효과 — 어떤 걸 골라야 할까?")
- 본문: 500~1500자, 소제목 3~5개
- 카테고리: 오메가-3 / 비타민 / 유산균 / 미네랄 / 복용법 중 택1
- 상태: "발행" 으로 설정 → nutriformula.co.kr에 자동 반영
- 하단에 CTA 블록: "나에게 필요한 영양제가 궁금하다면? → 무료 맞춤 분석 시작하기"

## 2단계 — 네이버 블로그 크로스포스팅

**누가:** 사장 (수동) 또는 Agent가 초안을 만들어 사장이 복붙
**어디서:** https://blog.naver.com (서박사 전용 네이버 블로그)
**규칙:**
- Notion 원본 글을 그대로 복사하되, 아래 광고 블록을 하단에 추가
- 이미지 1~2장 포함 (네이버 블로그 SEO에 이미지가 영향 큼)
- 태그: 주요 키워드 5~10개 (예: 마그네슘, 마그네슘효과, 영양제추천, 글리시네이트, 수면영양제)

## 3단계 — 인스타그램 카드뉴스

**누가:** Agent (Claude)
**도구:** `tools/instagram_card_generator.py`
**규칙:**
1. 블로그 글 내용을 기반으로 JSON 스펙 작성 (`.tmp/instagram/`)
2. 카드 구성:
   - 1장 (표지): 이모지 + 후킹 헤드라인
   - 2~9장 (본문): stat / list / body / quote 조합 (최대 8장)
   - 10장 (광고): 자동 삽입, 수정 불가
3. 테마 선택 기준:
   - 오메가-3, 생선 → ocean (파랑)
   - 비타민, 미네랄 → sage (녹색)
   - 면역, 피로 → terracotta (테라코타)
   - 여성건강, 피부 → berry (베리)
   - 복용법, 일반 → coral (코랄)
4. 출력: `대시보드/서박사의 영양공식/인스타/YYYY-MM-DD_주제.pptx`
5. 사장이 PPTX를 열어 PowerPoint에서 각 슬라이드를 이미지로 내보내기 → 인스타 업로드


## 4단계 — (향후) 유튜브/인스타 쇼츠

아직 미구축. 블로그 글 기반 60초 스크립트 자동 생성 Tool을 별도로 만들 예정.

## 채널별 계정 정보

| 채널 | 계정 | 상태 |

| nutriformula.co.kr | Vercel + Notion CMS | ✅ 운영중 |

## 주간 체크리스트

```
□ Notion 블로그 글 작성 완료
□ nutriformula.co.kr에 반영 확인
□ 네이버 블로그에 광고 포함 글 업로드
□ 인스타그램 카드뉴스 생성 + 업로드
□ (향후) 쇼츠 스크립트 + 영상 업로드
```
